# Modele Epidemiologiczne 🦠

Symulacja modeli epidemiologicznych do analizy rozprzestrzeniania chorób zakaźnych. 

## 📸 Podgląd działania
![Zrzut ekranu place holder]

## ✨ Funkcjonalności
- ✅ Implementacja różnych modeli epidemiologicznych (SIR, SEIR)
- ✅ Wizualizacja przebiegu epidemii
- ✅ Możliwość dostosowania parametrów symulacji

## 🚀 Instalacja i uruchomienie

### Wymagania
- Python 3.8+
- Biblioteki wymienione w `requirements.txt`

### Instalacja
```bash
# Klonowanie repozytorium
git clone https://github.com/wojciechbojakowski/Modele_epidemiologiczne.git
cd Modele_epidemiologiczne

# Instalacja zależności
pip install -r requirements.txt
```

### Uruchomienie symulacji
```
python main.py
```
